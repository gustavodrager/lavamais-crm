# Perguntas para Validar com o Major

## Operação atual

1. Como os clientes entram em contato hoje?
2. Quem responde o WhatsApp?
3. Quantas pessoas atendem clientes?
4. Existe algum sistema atual com cadastro?
5. Como ele sabe que um cliente ficou inativo?
6. Como ele controla aniversariantes?
7. Como ele identifica clientes VIP?
8. Como ele registra reclamações?
9. Como ele controla campanhas?
10. Como ele mede se uma campanha deu resultado?

## Cliente 360°

1. Quais informações são mais importantes no atendimento?
2. Quais preferências dos clientes costumam ser lembradas?
3. Quais informações hoje ficam só na cabeça dele?
4. Que tipo de alerta seria mais útil?
5. Quais dados não podem faltar no perfil do cliente?

## WhatsApp

1. Quais mensagens são enviadas com frequência?
2. Quais templates devem existir na primeira versão?
3. Quem pode enviar mensagens?
4. É necessário aprovar mensagens antes do envio?
5. Como a equipe sabe se o cliente respondeu?

## Campanhas

1. Quais campanhas ele gostaria de fazer primeiro?
2. Clientes inativos são prioridade?
3. Existe campanha de camisa social?
4. Existe campanha por bairro?
5. Existe campanha para delivery?
6. Existe campanha para cliente corporativo?

## Metas

1. Qual meta mensal de faturamento?
2. Existe meta diária?
3. Existe meta de clientes novos?
4. Existe meta de clientes recuperados?
5. Existe meta de peças?
