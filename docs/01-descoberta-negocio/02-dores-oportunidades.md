# Dores e Oportunidades

## Dores

| Dor | Impacto |
|---|---|
| Conhecimento sobre cliente está disperso | Atendimento depende da memória da equipe |
| WhatsApp concentra conversas | Histórico não fica estruturado |
| Clientes inativos não são acompanhados | Perda de recompra |
| Campanhas não são bem segmentadas | Baixa eficiência comercial |
| Falta de histórico de interação | Atendimento menos personalizado |
| Falta de metas visíveis | Menor controle comercial |

## Oportunidades

| Oportunidade | Valor gerado |
|---|---|
| Cliente 360° | Atendimento personalizado |
| Central de Relacionamento | Ação comercial diária |
| Segmentações automáticas | Recuperação de clientes |
| WhatsApp com template | Comunicação rápida |
| Histórico de interações | Memória compartilhada |
| Dashboard comercial | Gestão da unidade |
| Campanhas | Aumento de vendas |
