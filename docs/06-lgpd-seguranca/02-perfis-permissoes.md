# Perfis e Permissões

## Administrador

Acesso total.

Pode acessar:

- clientes;
- interações;
- notificações;
- campanhas;
- metas;
- relatórios;
- usuários;
- permissões;
- auditoria;
- configurações.

## Gerente

Pode acessar:

- clientes;
- interações;
- campanhas;
- metas;
- relatórios;
- notificações;
- Central de Relacionamento.

## Operador

Pode acessar:

- cadastro de clientes;
- Perfil 360°;
- movimentações comerciais;
- envio de notificações permitidas;
- registro de interações.

## Franqueadora

Acesso somente leitura a indicadores comerciais autorizados.

Sem acesso irrestrito a dados pessoais.
