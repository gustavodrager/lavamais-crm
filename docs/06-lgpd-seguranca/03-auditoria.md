# Auditoria

A auditoria deve registrar ações importantes realizadas no sistema.

## Eventos auditáveis

- login;
- criação de cliente;
- edição de cliente;
- inativação de cliente;
- envio de WhatsApp;
- criação de campanha;
- alteração de campanha;
- registro de interação;
- alteração de permissões;
- exportação de dados;
- alteração de metas.

## Dados do log

- usuário;
- ação;
- entidade afetada;
- data e hora;
- IP, se disponível;
- dados principais da alteração, quando aplicável.
