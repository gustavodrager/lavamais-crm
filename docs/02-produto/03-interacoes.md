# Interações

A entidade **Interação** registra relacionamento humano com o cliente.

Ela representa algo que a equipe sabe, ouviu, combinou ou registrou sobre o cliente.

## Tipos de interação

- WhatsApp;
- ligação;
- observação;
- reclamação;
- elogio;
- pedido especial;
- retorno de campanha;
- visita comercial.

## Exemplos

- Cliente viajará por 20 dias.
- Cliente reclamou de atraso na última entrega.
- Cliente prefere retirada pela manhã.
- Cliente pediu para não receber contato aos domingos.
- Cliente gostou da última campanha de camisa social.
- Cliente demonstrou interesse em plano corporativo.

## Campos sugeridos

- cliente;
- tipo de interação;
- resumo;
- descrição;
- sentimento;
- data;
- usuário responsável;
- próxima ação;
- data da próxima ação;
- status.

## Regra de experiência

Registrar uma interação precisa ser rápido.

Campos obrigatórios recomendados:

- tipo;
- resumo.
