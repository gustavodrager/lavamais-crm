# Cliente 360°

O Cliente 360° é o centro do LavaMais CRM e Relacionamento.

Ele deve reunir todas as informações relevantes para atendimento, relacionamento e ação comercial.

## Dados básicos

- nome;
- WhatsApp;
- telefone;
- e-mail;
- endereço;
- bairro;
- aniversário;
- origem;
- tipo de cliente.

## Dados comerciais

- cliente desde;
- última utilização;
- ticket médio;
- valor total gasto;
- quantidade de movimentações;
- frequência;
- tipo de atendimento mais comum;
- status comercial.

## Dados de relacionamento

- observações;
- preferências;
- restrições;
- reclamações;
- elogios;
- histórico de interações;
- informações importantes para atendimento.

## Comunicação

- notificações enviadas;
- mensagens abertas via WhatsApp;
- campanhas recebidas;
- respostas registradas;
- último contato realizado;
- usuário responsável pelo último contato.

## Classificações automáticas

- Ativo;
- Em risco;
- Inativo;
- Recuperado;
- VIP;
- Delivery;
- Corporativo;
- Aniversariante;
- Alto ticket;
- Queda de frequência.
